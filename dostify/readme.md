# Installation
## Pear-desktop
1. Install pear-desktop using <Javascript>yay -S pear-desktop</Javascript>
2. Open pear-desktop and setup API Plugin. Use port 6767
## Radio (MPV)
1. Install mpv using <Javascript>sudo pacman -S mpv</Javascript>
2. Start mpv server on "/tmp/pear-mpv.sock" using the command: "mpv --idle=yes --force-window=no --input-ipc-server=/tmp/pear-mpv.sock"
